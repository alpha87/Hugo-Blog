解决locale.Error: unsupported locale setting错误

Monday, March 18, 2019 • One minute to read

云服务器中使用pip工具的时候可能会报此类错误：

locale.Error: unsupported locale setting

是因为语言环境配置有误。

只需要在环境变量里添加这两行就可以解决这类问题。

如果你使用的是zsh，可以编辑.zshrc文件：

export LC_ALL=C.UTF-8
export LANG=C.UTF-8

保存后执行：

source ~/.zshrc

就可以啦！
